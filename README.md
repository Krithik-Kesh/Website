# Modern Personal Portfolio Website
A clean, responsive portfolio built with React and Tailwind CSS. It features a dynamic hero section, smooth scrolling navigation, an about section, skills grid, project gallery with hover effects, and a contact form. Fully mobile-friendly with light and dark mode support.

## Features
- Responsive design
- Smooth scroll navigation
- Dark/Light mode toggle
- Project showcase
- Contact form

## Technologies

- React
- Tailwind CSS
- Vite
- EmailJS
